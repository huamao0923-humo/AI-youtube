📋 YouTube API 設定步驟

1
建立 Google Cloud 專案
前往 console.cloud.google.com → 新增專案 → 啟用 YouTube Data API v3
2
建立 OAuth2 Desktop 憑證
APIs & Services → Credentials → Create Credentials → OAuth client ID → Desktop app → 下載 JSON
3
放置憑證檔案
將下載的 JSON 重新命名並放到：
config/youtube_client_secret.json
4
執行 OAuth2 授權（只需一次）
在本地執行，會自動開啟瀏覽器：
python -m modules.publish.youtube_uploader --auth-only
5
測試上傳
用最新腳本測試上傳一支私密影片（不公開）：
python daily_pipeline.py --upload --privacy private
