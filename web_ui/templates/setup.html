YouTube API 設定步驟
1
建立 Google Cloud 專案 + 啟用 YouTube Data API v3
前往 console.cloud.google.com → 新增專案 → APIs & Services → Enable APIs → 搜尋「YouTube Data API v3」並啟用
2
建立 OAuth2 Desktop 憑證
APIs & Services → Credentials → Create Credentials → OAuth client ID → Application type: Desktop app → 下載 JSON
3
放置憑證檔案
重新命名後放到：config/youtube_client_secret.json
4
執行 OAuth2 授權（只需一次）
在本地終端執行，會自動開啟瀏覽器授權：
python -m modules.publish.youtube_uploader --auth-only
5
測試上傳（私密影片）
python daily_pipeline.py --upload
上傳後到 YouTube Studio 確認影片出現，再改為公開。
