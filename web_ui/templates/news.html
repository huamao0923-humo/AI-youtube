💡 新手操作指南

  1. 每天早上先看「評分」 — 若有待評分，先執行 CoWork 評分：匯出 JSON → 交給 Claude Code → 匯入分數。ai_score ≥ 6 會自動標記為候選。
  2. 評分完成後開「聚類」 — 系統根據共同 entity（公司、技術）自動把相關新聞組成主題。你也可手動合併相似主題或拆分。
  3. 直接從「儀表板」Trending Topics 開集 — 點擊熱門主題 → 製作集數，是最快的路徑。「主題聚類」頁則是拿來深入檢視或調整分群。
  4. 「新聞池」是今天所有爬回來的新聞（含未評分、候選、已用），適合找漏網之魚手動推薦到 Brief。
  5. 「爬蟲日誌」是工具頁：看各日期爬取統計、按日清除舊資料。一般不需天天用。
